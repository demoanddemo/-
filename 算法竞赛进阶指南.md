### acwing 89 快速幂

求 a 的 b 次方对 p 取模的值。

输入格式
三个整数 a,b,p ,在同一行用空格隔开。

输出格式
输出一个整数，表示$a^b$ mod p的值。

数据范围
0≤a,b≤$10^9$ 
1≤p≤$10^9$
输入样例：

```
3 2 7
```

输出样例：

```
2
```

$10^9$的话C++ 会超时(TLE) 在 C++ 中 1 秒钟 可以算$10^6$到$10^8$次,所以要优化

>   即使是运算最快的C++，一秒种内的运算次数也只有10的8次方左右。而32位的int高达10的9次方
